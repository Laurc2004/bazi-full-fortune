# 常见陷阱与注意事项

本文档汇总了八字分析中容易出现的常见问题。分析时建议对照此清单，避免遗漏或误判。

---

## 大运与流年：建议用年份而非岁数

所有大运、流年、人生阶段的描述建议使用实际年份（如"2020-2029"），而非岁数（如"19-28岁"）。

原因：中国传统有虚岁和实岁之分，容易造成混淆。例如"30岁"可能是虚岁也可能是实岁，读者难以确定。

⚠️ 容易遗漏的位置：
- 身材走势（少年期/青年期 → 改为大运+年份）
- 事业时间线（年龄段 → 改为大运+年份）
- 财富等级预估中的"X岁后"
- 婚期最佳年中的"X岁"
- 感情模式中的"X-Y岁之间"
- 学业中的"X岁前"
- 精神世界中的大运年龄
- 综合特质总结中的"X-Y岁"

## 五行强弱判断：加权计算，避免简单计数

注意不要把地支和藏干本气重复计算。详见 SKILL.md "分析要点 → 五行强弱"。

加权参考：

| 来源 | 权重 |
|------|------|
| 天干透出 | 100%（1.0） |
| 月令本气 | 100%（1.0，当令加成） |
| 其他地支本气 | ≈70%（0.7） |
| 藏干中气 | ≈40%（0.4） |
| 藏干余气 | ≈20%（0.2） |

## 辛金喜忌：留意土重埋金

辛金（珠玉金）忌土重埋金。辛金喜壬水洗涤、喜甲木疏土。建议避免机械套用"印生身=印是喜用"的公式。

## 父母分析：避免套用"严父慈母"/"严母慈父"模式

详见 `references/family-patterns.md`。核心原则：先判断"在不在场"，再分析"严不严"。避免在没有校准事实的情况下直接套用刻板印象。

## cantian-tymext API：字段名为中文

`buildBaziFromSolar()` 返回的对象使用**中文**字段名。手写代码时需用中文字段访问：

```typescript
// ❌ 错误 — 这些字段不存在，返回 undefined
bazi.yearPillar
bazi.monthPillar
bazi.dayPillar
bazi.hourPillar

// ✅ 正确 — 中文字段名
bazi.年柱.天干.天干  // → "庚"
bazi.年柱.地支.地支  // → "辰"
bazi.月柱.天干.天干  // → "戊"
// ... 以此类推
```

完整字段结构示例（甲申年 戊辰月 甲寅日 庚午时）：
```json
{
  "性别": 1, "阳历": "...", "农历": "...", "八字": "...", "生肖": "猴", "日主": "甲",
  "年柱": { "天干": { "天干": "甲", "五行": "木", "阴阳": "阳", "十神": "比肩" }, "地支": { "地支": "申", ... } },
  "月柱": { ... }, "日柱": { ... }, "时柱": { ... },
  "胎元": ..., "胎息": ..., "命宫": ..., "身宫": ..., "神煞": [...], "大运": [...], "刑冲合会": {...}
}
```

### try/catch 静默吞错

`try/catch` 可能将 `TypeError: Cannot read properties of undefined` 静默吞掉，导致表面看到 0 matches 而不报错。编写 `scan_year` 类脚本时，建议避免将 pillar 访问放在宽泛的 try/catch 里——至少 log 一下非日期相关的错误。

> 经验教训：PR #2 修复了此 bug，原 `scan_year.ts` 用 `bazi.yearPillar.toString()` 导致所有反推扫描永远返回 0 matches。

## 六亲十神对应：务必区分男女命

男女命的六亲十神对应不同（详见 SKILL.md 第三章），这是八字论命的基础。常见失误：

1. 不分男女一律用"偏财为父、正印为母"
2. 不分男女一律用"正财为妻"——女命没有"妻"，女命配偶星是正官
3. 不分男女一律用"官杀为子女"——女命子女星是食伤，不是官杀

建议在分析开头明确标注命主性别对应的所有六亲十神，后续全文保持一致。

## 大运 ≠ 原命局：最常见的幻觉

分析十神时，建议严格区分"原命局（四柱）中有"和"大运中出现"。两者意义完全不同：

- 原命局有 = 命主先天自带的特质/缘分
- 大运有 = 某个10年阶段才会出现的运势/机遇

❌ 常见错误写法："庚金偏财不见天干，仅在巳中暗藏庚金（大运乙巳中才有）"
— 这混淆了大运和原命局。正确写法应分为两句：
1. "庚金偏财在原命局中完全不见（天干无、地支无、四柱所有藏干均无庚金）"
2. "仅在大运乙巳（2030-2039）中巳藏庚金才有偏财出现"

同理，正财辛金在原命局完全不见时，也不能写"暗藏"——如果四柱藏干中确实找不到，就是"完全不见"。

## 十神完整清单：不可遗漏藏干中的十神

建议在每个八字报告中列出"十神完整清单"，明确标注：
- 有哪些十神（含天干透出、地支本气、藏干中气/余气）
- 哪些十神完全不见（天干无、地支无、四柱所有藏干均无）

常见遗漏：
- 只看天干不看藏干 → 漏掉伤官、食神等暗藏十神
- 把藏干权重合计后描述错误（如 3×0.4=1.2 是"中等"而非"极弱"）
- "完全不见"清单漏项（如漏标偏财庚金）

## 空亡判断：建议以脚本输出为准

空亡（旬空）的判断建议直接从排盘脚本输出中读取，不建议手动推算。常见错误：
- 假设月柱不空亡，实际寅卯空落在寅上 → 月支空亡
- 空亡不等于"不在"，而是"占位减轻"、"影响力打折扣"

## 神煞分类：不要混淆不同神煞

童子煞 ≠ 华盖煞。两者含义不同：
- 童子煞：与佛道有缘、有灵性、前世修行
- 华盖煞：聪明、孤高、好文艺/宗教

在精神世界分析中，建议分别列出各自的神煞，不要混为一谈。

---

## "完全不见" vs "暗藏微弱"：务必核实

当日主的某个五行（如金）在四柱中完全不存在时，建议写"完全不见"而非"暗藏微弱"。

核实方法：逐一检查四柱所有藏干。例如午中藏丁（劫财）己（伤官），寅中藏甲（偏印）丙（比肩）戊（食神）——如果四柱所有地支的藏干中都没有庚辛金，那就是"完全不见"，不能说"暗藏"。

---

## 大运十神要完整列出

描述大运时，建议列出大运天干十神 + 大运地支所有藏干十神，不要遗漏。

例如丙火日主走乙巳大运：
- 乙 = 正印（天干）
- 巳藏丙 = 比肩、巳藏庚 = 偏财、巳藏戊 = 食神（地支藏干）
- 完整描述：正印 + 比肩 + 偏财 + 食神

❌ 常见遗漏：只写"正印 + 比肩 + 偏财"，漏掉食神。

## 输出格式：避免 Markdown 加粗和精确数字

- 某些渲染器不支持 `**加粗**`，建议用 Emoji 和结构来强调
- 身高、婚龄、金额等建议用范围描述，避免给出过于精确的数字
- 文件编码使用 UTF-8，中文输出
