# 家庭背景命理模式 / Family Background Patterns in Bazi

排盘后第一时间用来**校准家庭部分解读**的速查清单。每条模式都给出"命理信号 → 现实推断"和"实证校准问题"。

---

## 1. 父母离婚 / 父母缺席

### 命理信号清单（按权重）

| 权重 | 信号 | 解读 |
|---|---|---|
| ⭐⭐⭐ | 月柱天干冲日主 | 父母宫与子女缘分冲突/分离。最强信号。 |
| ⭐⭐⭐ | 月柱空亡 | 父母宫空缺，一个或双方"占位很轻"。 |
| ⭐⭐ | 偏财（父）/ 正印（母）不见天干 | 父母"不在场"。偏财不见 = 父不在；正印不见 = 母不在。 |
| ⭐⭐ | 偏财/正印只藏于余气（0.2） | 父母"几乎缺席"，可能物理上不在身边。 |
| ⭐ | 年支坐"胎"位 | 父亲漂泊、根基不稳（壬水坐午 = 父亲漂泊）。 |
| ⭐ | 壬丁合（年干合日干） | 父亲与日主精神亲密但物理距离远。 |
| ⭐ | 癸丁冲 | 母亲严厉但物理上缺席 / 母亲角色冲突。 |

### 优先级原则

1. **先回答"在不在场"再回答"严不严"** —— 空缺的父母谈不上性格
2. 多个信号叠加 = 离婚/缺席可能性**极高**
3. 单个信号（如偏财不见但月柱不冲）= 父亲远方/聚少离多，未必离婚

### 校准问题

- "父母是否在一起？"
- "如果分开，从小是谁带你？"
- "父亲现在常驻地在哪里？"
- "对母亲的印象深不深？"

### 反例（血泪教训）

拿到"壬午 癸丑 丁亥 己酉"后直接说"严父慈母"——实际上月柱癸冲日主丁 + 月柱空亡寅卯 + 偏财庚金不见 + 正印甲木藏亥中余气 = **父母离婚 + 母亲缺席 + 父亲远方**。

## 2. 父亲"远方"型

### 命理信号

- 偏财庚金（按主流：偏财为父）不见天干
- 偏财只藏于地支余气
- 年柱坐"胎"位（年支是父亲的"家"）
- 壬丁合 / 戊癸合（年干合日干）= 父亲与日主精神连但物理远
- 年柱纳音是"漂泊"型（如杨柳木、桑柘木）

### 现实推断

- 父亲**常年在外地工作**
- 聚少离多
- 父亲与日主关系**精神亲密但物理距离远**
- 父亲是"远方但懂你的人"

### 父亲性格信号（次要）

- 看正财辛金（如果出现）= 温润、细腻、有手艺
- 看正官/七杀（如果父亲星是壬/癸）= 父亲是"管教者"形象
- **先看在场度，再看性格**

---

## 3. 母亲"缺席"型

### 命理信号

- 正印甲木（按主流：正印为母）不见天干或只藏于余气
- 月柱天干冲日主（特别是癸丁冲）= 母亲严厉但物理缺席
- 月柱空亡在寅卯（印星位）= 母亲宫空缺
- 印星被食伤克（食神制印格局在母位）= 母亲被"压制"

### 现实推断

- 母亲**不在身边**（离婚、改嫁、出走、早亡）
- 即使存在，母亲也是 **"占位很轻"** 的角色
- 母亲可能严厉但**无法真正陪伴**
- 母亲角色需要通过其他渠道补位（伴侣、好友、长辈女性）

### 校准问题

- "对母亲的印象深不深？"
- "母亲在带你吗？还是其他长辈？"
- "母亲现在常驻地在哪里？"

---

## 4. 家里"开店"型（小个体户家庭）

### 命理信号

- 食伤（己土/戊土）旺
- 食神（己土）当令
- 食神制杀格局（食神温和管人）
- 食伤生财（靠手艺/服务换钱）

### 现实推断

- 家里做 **小本生意**：零售、餐饮、手工业、服务业
- 家庭氛围 **"店里为家"**
- 不是富裕家庭，但 **"有饭吃、不愁柴米"**
- 食神制杀 = 家里"以柔克刚"型氛围，**不闹到台面**

### 校准问题

- "家里做什么的？开店/体制内/务农/外出务工？"
- "如果是开店，做什么类型？"
- "家里的经济来源主要是？"

---

## 5. 父母关系"阴盛阳衰"型

### 命理信号

- 月柱食神（七杀位） = 食神制杀在父母宫
- 月柱天干是七杀、被食神制 = **母亲"制"住父亲**
- 偏财（父）不见 / 父亲角色弱

### 现实推断

- 母亲是 **家庭"轴心"**，父亲相对沉默
- 父母不是"恩爱型"，是"一个管一个"型
- 母亲操心多、付出多
- 父亲话少、退让

### 校准问题

- "父母中是不是有一方更操心？"
- "家里大事小事谁拍板？"

---

## 6. 母亲"控制欲"型（即使物理在场也控制强）

### 命理信号

- 月柱天干是七杀（癸水 / 乙木等）
- 七杀在父母宫
- 七杀被食神制（食神制杀）= 严厉被"藏"在温柔里
- 七杀冲日主（癸丁冲最常见）= 日主与母亲冲突

### 现实推断

- 母亲**外柔内刚** —— 外表温和，但**控制欲强**
- 母亲以"阴柔的方式"施压：唠叨、担忧、软性控制
- 日主与母亲 **冲突多**
- 母亲"为你好"是真的，但 **方式让人窒息** 也是真的

### 校准问题

- "小时候是不是母亲管得多、父亲相对自由？"
- "你和母亲关系怎么样？冲突多吗？"

---

## 7. 兄弟姐妹"缘薄"型

### 命理信号

- 比劫（丁火/丙火对丁火日主）力量弱
- 劫财（丙火）不见天干
- 比肩（丁火）只藏于一个地支

### 现实推断

- 家里 **可能就你一个**（独生子女概率高）
- 或者有兄弟姐妹但 **联系不紧密**、各自发展
- 即使有"发小/堂兄弟"补位，但 **都不算亲**
- 朋友多、知己少（比劫弱）

---

## 8. 命主"靠自己"型（无印护体）

### 命理信号

- 印星（甲木/乙木）弱
- 比劫（同类帮身）也弱
- 食伤旺（靠自己输出）
- 财星/官杀也不强

### 现实推断

- **天生靠自己** —— 没有"母亲庇护"或"家庭托举"
- 成熟得比同龄人早
- 靠手艺/知识吃饭
- 这种命一旦立住，**比有靠山的人更坚韧**

### 校准问题

- "从小到大是不是很多事要自己拿主意？"
- "家里对你的事业/学业有具体帮助吗？"

---

## 综合校准问题模板

排盘后立即向用户问：

```
帮你校准得更准一些，能告诉我以下信息吗？
1. 父母是否在一起？（离婚 / 一方缺席 / 双全）
2. 父亲现在做什么工作？常驻地在哪里？
3. 母亲在带你吗？还是由其他长辈带？
4. 家里做什么的？（开店 / 体制内 / 务农 / 外出务工）
5. 自己是做什么的？目前在人生哪个阶段？
```

不要堆在最后才问 —— 在"骨架分析"完成后**立即提问**，用户回答后再做完整解读。
