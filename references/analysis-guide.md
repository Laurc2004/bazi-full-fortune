# 一、校准工作流

不要拿到八字就直接给最终结论。八字是骨架，事实是血肉。第一轮分析如果不校准关键事实，可能全错。

## 排盘后必问的 5 个事实

1. 父母是否在一起 — 校准月柱天干冲日主的解读（离婚/分离信号）
2. 父亲职业/常驻地 — 校准父亲星是否「在场」
3. 母亲是否在带/是否缺席 — 校准母亲星是否「在场」
4. 家里做什么的（开店/体制内/务农/外出务工）— 校准食伤旺的解读
5. 自己的学历/工作状态 — 校准印星和财星的解读

## 工作流

1. 排盘得到八字 → 跑一遍「骨架分析」（五行强弱、格局、十神）
2. 立即向用户提出 5 个校准问题
3. 用户回答后重新分析那些命理信号
4. 给出最终报告

## 提问时机

不要堆在最后才问——在「骨架分析」完成后立即提问，用户回答后再做完整解读。

## 综合校准问题模板

```
帮你校准得更准一些，能告诉我以下信息吗？
1. 父母是否在一起？（离婚 / 一方缺席 / 双全）
2. 父亲现在做什么工作？常驻地在哪里？
3. 母亲在带你吗？还是由其他长辈带？
4. 家里做什么的？（开店 / 体制内 / 务农 / 外出务工）
5. 自己是做什么的？目前在人生哪个阶段？
```

# 二、五行强弱：加权计算

绝对不能把地支和藏干本气重复计算。

## 加权公式

| 来源 | 权重 |
|------|------|
| 天干透出 | 100%（1.0） |
| 月令本气 | 100%（1.0，当令加成） |
| 其他地支本气 | ≈70%（0.7） |
| 藏干中气 | ≈40%（0.4） |
| 藏干余气 | ≈20%（0.2） |

## 常见错误

❌ 地支有子水 + 子中藏癸水 = 算两次水
✅ 地支本气取一次权重，藏干中气/余气另算

## 实例演示

假设日主甲木，四柱为：甲寅 丙寅 甲子 丙寅

木的力量计算：
- 天干甲木 × 2 = 2.0
- 地支寅木本气 × 3 = 3 × 0.7 = 2.1
- 地支寅木藏干甲木（本气）已算过，不重复
- 地支寅木藏干丙火（中气）× 3 = 3 × 0.4 = 1.2（火，不算木）
- 地支子水本气 = 0.7（水，不算木）
- 地支子水藏干癸水（中气）= 0.4（水，不算木）

木总计 = 2.0 + 2.1 = 4.1（很强）

# 三、十神完整清单

每个报告必须列出完整的十神清单：

## 有哪些十神

- 天干透出的十神（权重最高）
- 地支本气的十神
- 藏干中气/余气的十神

## 完全不见的十神

- 天干无
- 地支无
- 四柱所有藏干均无

## 常见遗漏

❌ 只看天干不看藏干 → 漏掉伤官、食神等暗藏十神
❌ 把藏干权重合计后描述错误（如 3×0.4=1.2 是「中等」而非「极弱」）
❌ 「完全不见」清单漏项

# 四、大运十神要完整列出

列出大运天干十神 + 大运地支所有藏干十神，不要遗漏。

例如丙火日主走乙巳大运：
- 乙 = 正印（天干）
- 巳藏丙 = 比肩、巳藏庚 = 偏财、巳藏戊 = 食神（地支藏干）
- 完整：正印 + 比肩 + 偏财 + 食神

❌ 常见遗漏：只写「正印 + 比肩 + 偏财」，漏掉食神。

# 五、输出格式规则

1. 不用 Markdown 加粗（某些渲染器不支持）
2. 不给具体数字（身高/婚龄/金额等用范围描述）
3. 用 Emoji 做视觉分隔
4. 用 --- 做章节分隔
5. 文件编码 UTF-8
6. 中文输出
7. 所有年龄段/大运描述必须用实际年份（如 2020-2029），禁止用岁数（如 19-28岁），避免虚实岁混淆
